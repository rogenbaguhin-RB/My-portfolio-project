# My-portfolio-project
ITE-10 Portfolio Project

This is my ITE-10 project, a personal portfolio website built using HTML, CSS, and JavaScript.
The website showcases my skills, projects, and hobbies in a clean and user-friendly layout.

Features

Responsive design for desktop and mobile
Interactive navigation menu
Sections for About Me, Skills, Projects, and Contact

Technologies Used

HTML – for structure and content
CSS – for styling and layout
JavaScript – for interactivity

How to View

You can view the website by opening the index.html file in your browser or by visiting the live GitHub Pages link (if deployed).
